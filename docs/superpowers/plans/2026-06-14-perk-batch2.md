# 生存模式词条第二批（3 词条）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在已落地的 `SurvivalPerkManager` 之上新增 3 个生存词条——僵尸伤害 +5%/层（不限层）、僵尸前 10 次受击免伤/层（最多 2 层）、植物每 5 秒回 25 HP/层（最多 5 层，满层解锁过量治疗至 3×maxHealth）。

**Architecture:** 词条①是无状态倍率（镜像现有植物伤害词条，使用时缩放不写回 `mAttackDamage`）。词条②是 per-zombie 计数器（`Zombie::mFreeHitsRemaining`，出生初始化 + 随僵尸存档 + `TakeDamage` 首行消费）。词条③是 `Board` 全局脉冲计时器（`Board::UpdateLevel` 内每 5 秒遍历全部植物加血，封顶由词条规则决定）。数据/上限/规则全收进 `SurvivalPerkManager`；钟点只调用 getter。

**Tech Stack:** C++17、CMake + vcpkg（`clang-release` preset）、nlohmann/json、AutoTest JSON 脚本 + `dump_state` 闭环验证。

完整设计见 `docs/superpowers/specs/2026-06-14-perk-batch2-design.md`。

```cpp
int SurvivalPerkManager::ScaleZombieDamage(int base) const {
    return RoundScale(base, 1.0 + GetInfo(PerkType::ZOMBIE_DAMAGE_UP).perStack
                                 * GetStacks(PerkType::ZOMBIE_DAMAGE_UP));
}

int SurvivalPerkManager::GetZombieInvulnHits() const {
    return static_cast<int>(GetInfo(PerkType::ZOMBIE_INVULN_HITS).perStack
                            * GetStacks(PerkType::ZOMBIE_INVULN_HITS) + 0.5f);
}

float SurvivalPerkManager::GetPlantRegenInterval() const {
    return kPlantRegenIntervalSec;
}

int SurvivalPerkManager::GetPlantRegenPerPulse() const {
    return static_cast<int>(GetInfo(PerkType::PLANT_REGEN).perStack
                            * GetStacks(PerkType::PLANT_REGEN) + 0.5f);
}

int SurvivalPerkManager::GetPlantRegenHpCap(int maxHealth) const {
    // 满层（stacks 达 maxStacks）解锁过量治疗；判定用 >=maxStacks 而非硬写 ==5，调参自动跟随。
    if (GetStacks(PerkType::PLANT_REGEN) >= GetInfo(PerkType::PLANT_REGEN).maxStacks)
        return maxHealth * kPlantRegenOverhealMult;
    return maxHealth;
}
```

```cpp
	// 词条③：植物回血全局脉冲（生存专用；无词条→GetPlantRegenPerPulse()=0，整循环跳过，零开销）。
	// O(n) 遍历只在脉冲触发的那一帧发生（每 5s 一次），非每帧扫描。
	mPlantRegenTimer += deltaTime;
	if (mPlantRegenTimer >= mPerkManager.GetPlantRegenInterval())
	{
		mPlantRegenTimer = 0.0f;
		int heal = mPerkManager.GetPlantRegenPerPulse();
		if (heal > 0)
		{
			for (int id : mEntityManager.GetAllPlantIDs())
			{
				Plant* p = mEntityManager.GetPlant(id);
				if (!p || p->mIsPreview) continue;
				int cap = mPerkManager.GetPlantRegenHpCap(p->mPlantMaxHealth);
				if (p->mPlantHealth < cap)
				{
					int healed = p->mPlantHealth + heal;
					p->mPlantHealth = (healed > cap) ? cap : healed;
				}
			}
		}
	}
```
