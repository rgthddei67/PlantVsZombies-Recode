# 加固铁门僵尸设计

## 定位与数值

- 类型名：`ZOMBIE_REINFORCED_DOOR`，图鉴名“加固铁门僵尸”。
- 复用铁门僵尸动画、死亡/啃食帧事件与手臂状态机，不增加帧事件。
- 本体生命 500，铁门生命 1000。
- 进入生存随机池：`weight=2100`、`survivalRound=6`。后续按主人要求加入冒险：2-8 用普通铁门作家族对照独立教学，2-9 必进 8 种综合池。
- 正式波次每波至多生成 2 只；第三次及以后抽到时回退为普通铁门僵尸。直造、图鉴和开发者工具不受此限制。

## 伤害契约

- 大喷菇与寒冰大喷菇对其伤害先乘 2，再进入词条缩放和下述上限。
- 铁门仍在时，来自植物的普通伤害按最终实际伤害每次最多 15；大喷穿透时门与本体各承受同一笔 15 点伤害。
- 灰烬/爆炸伤害无论铁门是否存在，每次最终实际伤害最多 100，且该僵尸永不进入化灰表现。
- 植物直杀不能绕过生命值：大嘴花咬杀尝试改为 15 点植物普通伤害；土豆雷走灰烬伤害入口，最多 100。
- 小推车保持 `DamageSource::OTHER + INT32_MAX` 的既有路径，不受植物伤害上限影响。

## 大喷阻断与视觉

- 大喷攻击先收集本行有效目标并按世界 X 从近到远排序，避免依赖实体容器迭代顺序。
- 加固铁门在本次命中开始时仍持门，则自身正常承伤，随后停止结算，门后的僵尸不受伤也不吃寒冰追加效果；即使命中恰好打掉门，本次喷雾仍被挡住。
- 门掉后不再阻断后续喷雾，但双倍伤害仍生效。
- `FumeCloud`/`IceFumeCloud` 仍使用原配置；运行期给粒子特效附加世界坐标右裁剪边界，取阻断僵尸碰撞框前沿。无需按距离重建 XML 或预先设置一套离散长度。

## 视觉资源

- 只重着色铁门三张破损阶段贴图，选择高饱和青绿色工业漆；保留原 alpha、明暗、锈蚀、高光、孔洞和描边。
- `scripts/recolor_reinforced_door.ps1` 同源生成两个 preset 的贴图；掉门粒子使用第三阶段的同色贴图。

## 验证

- AutoTest 覆盖：基础血量、持门普通伤害上限、大喷双倍与前后排阻断、门掉后恢复穿透、灰烬 100/免化灰、大嘴花/土豆雷直杀免疫、小推车例外、每波 2 只上限与新波重置。
- 截图至少包含完整配色、持门喷雾裁断、掉门后喷雾贯穿和灰烬攻击后仍存活。
- `clang-release` 配置与构建零警告；从 `build/clang-release/` 在当前桌面可见运行测试，核对退出码、`run.log`、状态 JSON 和截图。
