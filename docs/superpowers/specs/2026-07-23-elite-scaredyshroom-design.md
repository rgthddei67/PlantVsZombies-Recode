# 2-8 精英胆小菇设计

## 目标

2-8 通关奖励一株独立的紫色精英胆小菇，作为后续高压关卡的可成长远程主力。它以“长时间保持安全距离”换取高后期输出；近身受惊会清空全部成长，避免成为无条件超模射手。

## 数值

- 阳光 225，冷却 20 秒，生命 500。
- 初始孢子伤害 `a = 10`。
- 每 `n = 5` 发获得一层攻速，每层把射击间隔乘 `0.85`，即 `b = 15%` 间隔缩减；初始间隔 1.5 秒，最快 0.2 秒一发。
- 每 `c = 10` 发使孢子伤害增加 `d = 1`，最高 15。
- 白天不睡觉，但每次发射只获得 0.6 发成长，即成长速度降低 `e = 40%`。
- 从 READY 首次进入害怕状态时，攻速与伤害成长立即归零；害怕期间射击计时仍由原胆小菇状态机压回零。

夜间连续输出约 39 秒升到 15 伤，约 43 秒触及最快射速；白天达到同阶段所需实际发数约为夜间的 1.67 倍。

## 实现

- 新类型 `PLANT_ELITE_SCAREDYSHROOM` 追加在 `PlantType` 末尾，避免旧存档整数 ID 错位。
- `EliteScaredyShroom : ScaredyShroom` 复用已经验证的第 25 帧吐弹事件与四态害怕状态机，不新增帧事件。
- 基类提供射击间隔、孢子伤害、成功发弹和受惊起点四个小型扩展入口；普通胆小菇保持 20 伤 / 1.5 秒原行为。
- 成长进度进入植物额外存档；AutoTest 输出成长发数、攻速阶段、当前伤害、射击间隔和昼夜成长率。
- 实战与预览使用紫色 overlay；卡片由原胆小菇卡图的可复现 PowerShell 脚本生成独立紫色版本。
- 2-8 对应内部关卡 17；奖励表把该关原本的无植物奖励替换为精英胆小菇。

## 加固铁门结论

以 2026-07-23 当前源码为准：本体 300、门 920、持门植物普通伤害单次上限 10、灰烬上限 320。精英胆小菇从零成长、全程不受惊时仍需约 48 秒连续射击才能打破 920 门血，随后才以最高 15 伤攻击本体。当前门血已经能约束成长射手，不再上调；提高门血会进一步挤压原版植物。

## 验证

- 夜间：成长到 15 伤与 0.2 秒射速，近身受惊后五项观测值回到初始状态。
- 白天：保持清醒，成长率投影为 60%。
- 奖励：以真实奖杯点击流程结算 2-8，验证卡牌进入拥有列表。
- 视觉：与普通胆小菇和小喷菇同行截图，校对脚底、影子和紫色识别度。
- 子弹对象池：复位自定义伤害与速度，防止精英孢子污染后续普通子弹。
