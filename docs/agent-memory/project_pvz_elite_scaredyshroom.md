# 精英胆小菇

## 当前契约

- 类型 `PLANT_ELITE_SCAREDYSHROOM` 追加在旧植物枚举末尾，冒险显示关卡 2-8（内部 level 17）通关奖励；卡片 225 阳光、20 秒冷却。
- `EliteScaredyShroom : ScaredyShroom` 复用既有第 25 帧吐弹事件与四态害怕状态机。生命 500；初始孢子伤害 10，每 10 发 +1，最高 15；初始间隔 1.5 秒，每 5 发间隔缩短 15%，最低 0.2 秒。成长进度最多 65。
- 受惊开始时成长进度清零，因此攻速、伤害都回到初始值。白天不睡觉，但每次成功吐弹只增加 0.6 成长，较夜间慢 40%；小数进度参与存档。
- 每个关卡累计最多种植 3 株，死亡、铲除或被台风吹走都不返还次数；其余战斗数值不变。累计数由 `Board` 保存，旧档无字段时以存活精英胆小菇数作为保守下界。
- 游戏内复用胆小菇 reanim 并施加紫色 overlay，卡片使用 `scripts/recolor_elite_scaredyshroom.ps1` 从原卡图可重复生成独立紫色资源。

## 高速射击约束

- 射击间隔不能直接等同于整段动画重播速度。`mShotPending` 从起播保持到第 25 帧吐弹事件，事件真正创建孢子后才允许下一次重播；否则 0.2 秒档会在到达帧事件前反复重启动画，表现为“最后几下很快但没有子弹”。
- 高速动画只需把 16→25 帧约 0.75 秒的吐弹前摇压进当前间隔，速度为 `attackSpeedMultiplier * max(1.5, 0.75 / interval)`，不要按完整 1.5 秒基准成倍提速。
- 害怕守卫只覆盖吐弹前的 pending 阶段；不能屏蔽整段 `anim_shooting`，否则 0.2 秒连续重播会让植物永久无法受惊。
- 目标缓存刷新为 0.1 秒并与射击计时独立，保证最低 0.2 秒间隔不会额外被索敌节流拖慢。

## 验证

- 桌面可见 `smoke_elite_scaredyshroom.json` 覆盖 500 生命、65 成长、13 攻速阶段、15 伤、200ms、同屏多发孢子、受惊全清、白天单发增长 0.6，以及与普通胆小菇/小喷菇的视觉站位。本次三株限制按主人要求未运行 AutoTest，仅做编译验证。
- `smoke_adventure_progression.json` 覆盖 2-8 奖杯领取后 `adventureLevel=18` 且精英胆小菇卡片归属成立。
- AutoTest 的普通存档路径仍有短路；成长后的手动退出、重进读档需由主人做最终验收。
